== Description ==\
\
THe pluing allows you to add a follow button to your website, allowing a user to follow your tumblr account.\
\
 \
\
After entering your tumblr username you can choose to use either a light or dark button, this can then be added as a widget to your website.\
\
== Installation ==\
\
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.\
